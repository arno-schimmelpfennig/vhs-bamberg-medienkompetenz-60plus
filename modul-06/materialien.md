# Modul 6 Materialien
## Mein digitaler Werkzeugkasten - Rueckblick, Sicherheit und naechste Schritte

### Meine persoenliche Sicherheits-Checkliste

Kreuzen Sie an, was Sie koennen. Feiern Sie jeden Haken!

**Grundlagen (Modul 1):**
- [ ] Ich kann mein Geraet ein-/ausschalten und entsperren
- [ ] Ich kenne meinen Homescreen und kann Apps finden
- [ ] Ich kann eine App installieren

**Sicherheit (Modul 2):**
- [ ] Ich habe eine Bildschirmsperre eingerichtet
- [ ] Ich verwende sichere Passwoerter (Satz-Methode)
- [ ] Ich habe App-Berechtigungen geprueft

**Kritisches Denken (Modul 3):**
- [ ] Ich kenne den 5-Finger-Check fuer verdaechtige Nachrichten
- [ ] Ich erkenne Phishing-E-Mails
- [ ] Ich habe ein Familien-Codewort vereinbart

**Kommunikation (Modul 4):**
- [ ] Ich kann WhatsApp-Nachrichten senden und empfangen
- [ ] Ich kann einen Videoanruf starten
- [ ] Ich habe meine Datenschutz-Einstellungen geprueft

**Alltagsgeschaefte (Modul 5):**
- [ ] Ich weiss, wie Online-Banking sicher funktioniert
- [ ] Ich kenne die E-Rezept-App
- [ ] Ich habe die Bamberg App installiert

**Meisterschaft (Modul 6):**
- [ ] Ich kann die 3 Goldenen Regeln auswendig
- [ ] Ich kann jemandem erklaeren, wie man sicher im Internet ist
- [ ] Ich habe einen Plan, wie ich weiter dazulerne
